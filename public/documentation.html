Movie API

API will return list of all movies to the user, as well as data about genre, and directors.
API will also allow users to register/creaet accounts, update user info, add/remove movies to favorites, and allow existing users to deregister. REQUESTS/ ENDPOINTS: 1. Return a list of ALL movies to the user: URL: /movies Method: GET Query Parameters: none Request format: text Response format: JSON 2. Return data (description, genre, director, image URL, whether itâ€™s featured or not) about a single movie by title to the user: URL: /movies/:title Method: GET Query Parameters: Movie Title (ex: The Dark Knight) Request format: text (ex: /The%20Dark%20Knight) Response format: JSON 3. Return data about a genre (description) by name/title: URL: /genres/:name Method: GET Query Parameters: Genre name (ex: Action) Request format: text (ex: /Action) Response format: JSON 4. Return data about a director (bio, birth year, death year) by name: URL: /directors/:name Method: GET Query Parameters: Director name (ex: Christopher Nolan) Request format: text (ex: /Christopher%20Nolan) Response format: JSON 5. Allow new users to register: URL: /users Method: POST Query Parameters: none Request format: json (ex: username: johndoe123 password: password123 email: john123@doe.com) Response format: text 6. Allow users to update their user info (username): URL: /users/:username Method: PUT Query Parameters: username (ex: /johndoe123) Request format: json (ex: username: coolguy123) Response format: text 7.
